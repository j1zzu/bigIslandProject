# Слои проекта QGIS

| Слой | Назначение | Публикация |
|---|---|---|
| `zones` | Зоны острова | WMS, WFS |
| `revit_boundaries` | Проверенные импортированные контуры | WMS, WFS |
| `revit_points` | Проверенные координатные точки | WMS, WFS при необходимости |

Базовая карта загружается Leaflet из OpenStreetMap, поэтому публиковать её через QGIS не требуется. CRS проекта QGIS — `EPSG:32653`. Сервер должен разрешать выдачу WMS в `EPSG:3857`, а WFS — в `EPSG:4326`.
